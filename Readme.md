OTUS, Урок 5, Flask
Магазин товаров для животных


Необходимо установить Flask:
pip install flask

Описание
1. Файл likedatabase.py содержит каталог товаров в виде
 
    dogs_feed = (
        ('Сухой корм для взрослых собак', '350', '120', 1),
        ('Сухой корм для щенков', '400', '2000', 11),
        ('Консервы для щенков', '100', '150', 87  ),
    )
    
    где 1-й элемент Наименование продукта,
        2-й элемент цена(руб)
        3-й элемент количество на складе(шт)
        4-й элемент id

Также содержит перечень категорий товаров:

   category = ('Все', 'Корма для собак', 'Корма для кошек', 'Корма для хомячков')

2. index_views.py - вывод главной страницы и страниц категорий
3. products_views.py - вывод характеристик конкретного товара
